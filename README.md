# Telecom Customer Churn / Classification Project

This repository contains data and notebooks for a telecom customer dataset. It includes cleaning and training notebooks along with helper scripts and CI to make the project ready for GitHub.

Contents
- `cleaned.csv`, `dataset.csv` - dataset files
- `cleaning.ipynb` - data cleaning notebook
- `training.ipynb` - model training notebook
- `data_inspect.py` - small CLI to inspect datasets

Getting started

1. Create a Python virtual environment and install dependencies:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```


2. Open the notebooks with Jupyter Lab / Notebook:

```bash
jupyter lab
```
